# XGBoost Studio 需求文档 v2.1
> 本版本对标 **JMP Pro 16** 专业统计分析平台，从数据分析师视角补强统计严谨性、交互式诊断、模型解释深度与预测不确定性量化。

---

## 一、项目概述

| 项目 | 内容 |
|------|------|
| **产品名称** | XGBoost Studio |
| **产品类型** | 跨平台桌面应用（本地 C/S 架构） |
| **目标用户** | 数据分析师、算法工程师、科研人员 |
| **核心价值** | 无需编写代码，全流程可视化完成 XGBoost 建模、调优、解释与预测；内置特征分析、模型管理与交互式预测，关注性能、可解释性与用户体验 |

---

## 二、技术架构

```
客户端：Electron 28 + React 18 + TypeScript + Ant Design 5 + ECharts
服务端：Python 3.11 + FastAPI + XGBoost + Optuna + SHAP + Pandas + Scipy + Statsmodels + Scikit-learn
通信：HTTP REST + SSE（实时推送训练进度与调优进度）
打包：Electron Builder（客户端）+ PyInstaller（服务端）
本地存储：SQLite（项目元数据、模型注册表）+ 文件系统（模型文件、报告缓存）

关键库职责说明：
  Scipy / Statsmodels  ── 分布拟合检验、ANOVA、VIF、Box-Cox 变换
  Scikit-learn         ── PCA、校准曲线、学习曲线、交叉验证、基线模型
  SHAP                 ── SHAP 值、PDP、ICE、交互效应图
  Optuna               ── 贝叶斯/随机/网格/拉丁超立方超参数搜索
```

## 四、功能模块详细设计

### 模块 1：数据导入与预览

| 功能点 | 描述 |
|--------|------|
| 文件导入 | 支持 `.xlsx` / `.xls` / `.csv` 格式，拖拽或点击上传 |
| 多 Sheet 选择 | 若 Excel 有多个 Sheet，可下拉选择，实时预览 Sheet 内容 |
| 数据预览表格 | 虚拟滚动分页展示原始数据，支持列宽调整、列固定、全屏模式 |
| 数据概览统计 | 每列的：数据类型、非空数量、缺失率、均值/中位数/标准差/最大/最小值、唯一值数量 |
| 列数据分布图 | 点击列名弹出该列的直方图/饼图（数值型/类别型自动切换） |
| 缺失值处理 | 可选：均值填充 / 中位数填充 / 众数填充 / 常量填充 / 删除行 / 不处理；支持按列单独配置 |
| **缺失值模式分析** | **热力图展示各行各列缺失值的共现模式（对标 JMP 缺失值分析平台），辅助判断缺失机制（MCAR/MAR/MNAR）；计算列间缺失值相关性，识别"同时缺失"的特征组** |

| 重复行检测 | 自动统计重复行数量，一键删除或查看重复明细 |
| 数据质量评分 | 综合缺失率、异常率、重复率给出 0–100 的数据质量评分及改善建议 |

### 模块 2：特征分析（新增）

> **设计目标**：在建模前全面理解数据，发现冗余特征、强相关特征与目标变量之间的关系，辅助特征筛选决策。

#### 2.1 单特征分析

| 功能点 | 描述 |
|--------|------|
| 分布可视化 | 数值型：直方图 + KDE 曲线；类别型：条形图 + 频率表 |
| **分布拟合检验** | **自动拟合常见分布（正态、对数正态、指数、均匀等），使用 Shapiro-Wilk / Anderson-Darling / KS 检验评估拟合优度，给出 p 值；告知用户数据是否符合正态假设（对标 JMP Distribution 平台）** |
| **变换建议** | **对偏度较高的数值特征，自动搜索 Box-Cox 最优 λ 值（或 Yeo-Johnson，支持负值），给出"变换前 vs 变换后"分布对比图及偏度改善程度，供用户决策是否应用** |
| 偏度/峰度指标 | 显示偏度（Skewness）、峰度（Kurtosis），提示是否需要变换 |
| 箱线图 | 展示四分位、中位数、异常点 |
| 与目标变量关系 | 数值特征：散点图 + 趋势线 + Pearson/Spearman 相关系数与 p 值；类别特征：分组箱线图，附 ANOVA F 值与 p 值标注 |

#### 2.2 多特征关联分析

| 功能点 | 描述 |
|--------|------|
| 相关性热力图 | Pearson / Spearman / Kendall 三种相关系数可切换，颜色深浅直观表示相关强度，支持按相关系数排序聚类展示 |
| 高相关性预警 | 自动标识相关系数绝对值 > 阈值（默认 0.85）的特征对，建议考虑删除其中一个 |
| **VIF 多重共线性检测** | **计算每个特征的方差膨胀因子（Variance Inflation Factor），VIF > 10 时红色预警，VIF 5~10 橙色提示；排名表格直观展示，建议对高 VIF 特征做处理（对标 JMP Multivariate 平台）** |
| 散点矩阵 | 支持选择多列（最多 8 列）生成 Pair Plot，对角线为分布图，上三角为相关系数，下三角为散点图 |
| 类别特征与数值特征交叉 | 分组箱线图展示类别特征各组在数值特征上的分布差异，附 Levene 方差齐性检验 |
| 目标相关性排名 | 计算每个特征与目标列的相关系数并排序，快速定位强相关特征 |
| 方差分析（ANOVA） | 类别特征对目标变量的 F 值和 p 值，辅助判断特征显著性，附 Tukey HSD 事后多重比较 |
| 互信息排名 | 使用互信息（Mutual Information）评分评估每个特征对目标的非线性相关度 |

#### 2.3 类别不平衡分析

| 功能点 | 描述 |
|--------|------|
| 类别分布图 | 饼图 + 数量统计，直观展示各类别比例 |
| 不平衡预警 | 当最大类别/最小类别样本比 > 阈值时，自动给出提示及建议（调整 `scale_pos_weight`） |

#### 2.4 主成分分析（PCA）辅助

> 对标 JMP Principal Components 平台，辅助用户理解特征结构、发现冗余维度。

| 功能点 | 描述 |
|--------|------|
| Scree Plot | 各主成分解释方差比例折线图（碎石图），附累计方差曲线，辅助选择合适的主成分数量 |
| 载荷图（Loadings） | 展示每个原始特征在各主成分上的载荷值，识别哪些特征"打包"在一起 |
| 双标图（Biplot） | 同时展示样本得分与特征载荷，直观感受特征与样本的分组关系 |
| 降维建议 | 给出"保留 N 个主成分可解释 X% 方差"的建议，供用户决策是否在特征工程阶段使用 PCA 降维 |

### 模块 3：特征工程与选择

| 功能点 | 描述 |
|--------|------|
| 因变量（目标列）选择 | 下拉单选，自动识别任务类型（分类/回归），支持手动覆盖 |
| 自变量（特征列）选择 | 多选勾选，支持全选/反选；结合模块 2 的相关性排名与 VIF 辅助选择；支持按"重要性建议"列表一键批量勾选 |
| 任务类型确认 | 自动推断 + 手动覆盖（二分类 / 多分类 / 回归） |
| 特征类型标注 | 自动识别数值型/类别型，支持手动修正 |
| 类别特征编码 | 可选：One-Hot 编码 / Label 编码 / 频率编码 / 不处理 |
| 数值特征变换 | 可选：对数变换 / **Box-Cox（自动搜索最优 λ）** / **Yeo-Johnson（支持负值，自动搜索最优 λ）** / 标准化（Z-score）/ 归一化（Min-Max）/ 不处理；按列单独配置，并在配置页展示变换前后的分布对比 |

| 随机种子设置 | 可输入固定随机种子，保证可复现 |
| 划分结果预览 | 显示各集合的样本数量及目标分布（分类任务显示各类别占比，检验分层是否合理） |

### 模块 6：模型评估（增强）

> **设计目标**：不仅展示指标数值，还提供诊断视角，帮助用户判断模型是否过拟合、是否达到基线要求、在哪些样本上表现差。

#### 6.1 性能指标看板

| 任务类型 | 指标 |
|---------|------|
| 二分类 | Accuracy, Precision, Recall, F1, AUC-ROC, AUC-PR, Log Loss, MCC（Matthews 相关系数） |
| 多分类 | Accuracy, Macro/Micro/Weighted F1, Cohen's Kappa, Confusion Matrix（支持归一化） |
| 回归 | RMSE, MAE, MAPE, R², Adjusted R², 最大误差, 误差百分位分布（P50/P90/P99） |

**指标看板交互：**
- 每个指标旁边提供"好/中/差"的直观评级（基于领域经验阈值）
- 指标数值支持与训练集/验证集/测试集三套数据横向对比，一眼判断过拟合
- **所有指标附带 95% Bootstrap 置信区间**（对标 JMP 统计严谨性），帮助用户判断指标差异是否具有实际意义
- 点击指标卡片可下钻查看详细分布

#### 6.2 交叉验证详情

> 对标 JMP Pro 交叉验证报告，不仅给出均值，还展示稳定性。

| 功能点 | 描述 |
|--------|------|
| K-Fold 逐折结果表格 | 展示每个 Fold 的训练集/验证集指标数值，最后一行为均值 ± 标准差 |
| Fold 间方差评估 | 若标准差过大（相对于均值），给出"模型不稳定"警示，建议增大 K 或检查数据分布 |
| 箱线图 | 将所有 Fold 的指标绘制为箱线图，直观展示稳定性分布 |

#### 6.3 过拟合诊断

| 功能点 | 描述 |
|--------|------|
| 训练-验证差距指示 | 自动计算训练集与验证集关键指标差值，差距超过阈值时用红色警示 |
| 学习曲线 | 绘制不同训练样本量下的训练/验证分数，诊断数据量是否足够 |
| 验证曲线 | 绘制关键参数（如 `max_depth`、`n_estimators`）取不同值时的训练/验证分数变化，辅助调参 |

#### 6.4 分类任务专项

| 功能点 | 描述 |
|--------|------|
| 混淆矩阵 | 热力图展示，支持切换绝对数量/归一化百分比 |
| ROC 曲线 | 含 AUC 值标注；多分类支持 One-vs-Rest 的多条 ROC 曲线；**可拖动阈值点实时更新混淆矩阵** |
| PR 曲线 | Precision-Recall 曲线，适用于类别不均衡场景，附 Average Precision 值 |
| **提升图 / 累积增益图** | **Lift Chart 和 Gains Chart（对标 JMP 模型评估）：按预测概率排序后，展示在 Top-X% 样本中覆盖正样本的比例，直接体现模型的业务价值（如：触达 20% 用户可覆盖 60% 转化）** |
| **校准曲线（可靠性图）** | **将预测概率按分桶展示实际正样本率 vs 预测概率均值（对标 JMP Reliability Plot），完美校准应贴近对角线；同时显示 Brier Score；提示模型是否存在概率过高/过低的系统性偏差** |
| 分类阈值调节 | 二分类任务：可拖动阈值滑块，实时查看 Precision / Recall / F1 的变化，找到最优决策边界；**支持设置不同误分类代价（Cost Matrix）来自动推荐最优阈值** |
| 错误样本分析 | 列出被误分类的样本，显示其特征值、真实标签与预测标签，支持按置信度排序 |
| 预测概率分布图 | 正负样本的预测概率分布直方图（校准图），评估模型置信度是否合理 |

#### 6.5 回归任务专项

| 功能点 | 描述 |
|--------|------|
| 残差图 | 预测值 vs 真实值散点图，理想情况应随机分布在水平线附近；**自动添加 LOESS 平滑曲线检测系统性偏差** |
| 残差分布图 | 残差的直方图 + QQ 图，检验残差是否符合正态分布，附 Shapiro-Wilk 检验 p 值 |
| **预测区间（Prediction Interval）** | **使用 Bootstrap 方法计算每个预测值的 95% 预测区间，在结果表格和图表中展示不确定性范围（对标 JMP 预测区间）** |
| 大误差样本分析 | 列出绝对误差最大的 Top-N 条样本，供用户人工排查原因 |
| 预测 vs 真实对比图 | 折线图展示排序后真实值与预测值的拟合程度 |

#### 6.6 基线对比

| 功能点 | 描述 |
|--------|------|
| 简单基线自动计算 | 自动计算均值预测/众数预测（DummyClassifier/DummyRegressor）的指标作为基线参考 |
| 基线对比表格 | 当前模型指标与基线并排展示，直观显示模型带来的提升 |
| **模型对比统计检验** | **多个模型互相对比时，提供 McNemar 检验（分类）或 Diebold-Mariano 检验（回归）的 p 值，告知用户指标差异是否具有统计显著性，而非随机波动（对标 JMP 统计严谨性）** |

### 模块 7：模型调优（增强）

> **设计目标**：对标 JMP Pro 的 Profiler + DOE 参数搜索理念——调优不仅是"跑更多次试验"，更要让用户**理解参数空间的结构**，知道哪些参数重要、参数间如何交互、最优解在哪个区域。

#### 7.1 调优策略

| 策略 | 描述 |
|------|------|
| 贝叶斯优化（推荐） | 基于 Optuna TPE 算法，建立概率代理模型智能采样，效率最高 |
| **拉丁超立方采样（空间填充设计）** | **对标 JMP Computer Experiment DOE：在参数空间中均匀布点，确保每个维度都被覆盖，适合首次探索整体参数响应面，避免贝叶斯初期的聚集性偏差** |
| 随机搜索 | 随机采样参数组合，适合快速粗筛 |
| 网格搜索 | 穷举指定参数网格，适合小范围精细调优 |
| 交叉验证调优 | 支持 K-Fold 交叉验证评估，提升评估稳定性（可设置 K 值） |

#### 7.2 调优配置

| 功能点 | 描述 |
|--------|------|
| 参数搜索空间设置 | 每个参数可设置搜索范围（数值型：min/max/step；类别型：候选列表），支持对数尺度搜索（适合 `learning_rate`） |
| 优化目标指标选择 | 可选择优化的指标（如 F1、AUC、RMSE 等），支持最大化/最小化 |
| **多目标优化** | **支持同时优化两个指标（如最大化 AUC 同时最小化训练时间），生成 Pareto 前沿曲线，由用户从帕累托最优解中手动选择偏好点** |
| 搜索预算设置 | 设置最大试验次数 / 最长调优时间（分钟） |
| 提前停止策略 | 若连续 N 次试验无改善，提前终止搜索 |

#### 7.3 调优过程监控

| 功能点 | 描述 |
|--------|------|
| 实时试验进度 | SSE 推送，显示当前第几次试验/总次数，进度条 + 预计剩余时间 |
| 实时最优得分曲线 | 折线图显示随试验次数增加最优得分的变化趋势 |
| 试验历史表格 | 每次试验的参数组合与得分，支持排序、筛选、导出 CSV |
| 中止调优 | 支持随时中断调优，保留已发现的最优参数 |

#### 7.4 调优结果分析（对标 JMP Profiler 体系）

| 功能点 | 描述 |
|--------|------|
| 参数重要性图 | 展示各参数对模型性能影响的重要性（Optuna 内置 FAnova），直方图排名展示 |
| **预测廓线图（Prediction Profiler）** | **核心功能，对标 JMP Profiler：为每个超参数绘制一条"单变量响应曲线"（固定其他参数时该参数与评估指标的关系），用户可拖动滑块交互式调整单个参数值，其他图实时联动更新，直观感受各参数的影响方向和强度** |
| **等高线廓线图（Contour Profiler）** | **对标 JMP Contour Profiler：选择任意两个超参数，生成二维热力图（X轴为参数1，Y轴为参数2，颜色为评估指标），直观展示两参数的联合效应与交互区域；最优区域自动用等高线圈出** |
| **参数交互效应图** | **当两个参数存在显著交互时（FAnova 检测），绘制交互热力图或 3D 曲面图，提示用户"这两个参数需要协同调整"** |
| 参数与得分散点图 | 每个参数取值 vs 对应得分的散点图，辅助判断最优区间 |
| 最优参数一键应用 | 将调优发现的最优参数自动填入参数配置面板，一键触发重新训练 |
| 调优记录持久化 | 调优结果保存到当前项目，下次打开可继续查看历史调优记录，支持"断点续调" |

### 模块 9：分析报告

#### 9.1 可视化图表

| 图表 | 说明 |
|------|------|
| 训练曲线 | Train/Val Loss 曲线，含早停标记 |
| 混淆矩阵 | 热力图展示（分类任务） |
| ROC / PR 曲线 | 含 AUC 值标注（分类任务） |
| 提升图 / 增益图 | 按预测概率排序的业务价值评估图（分类任务） |
| 校准曲线 | 预测概率可靠性图（分类任务） |
| 残差图 | 预测值 vs 真实值散点图（回归任务） |
| 特征重要性 | XGBoost 内置：weight / gain / cover 三种模式可切换，支持 Top-N 筛选 |
| **部分依赖图（PDP）** | **对标 JMP Pro PDP：选择 1~2 个特征，固定其他特征为均值，绘制该特征取值变化时模型预测均值的变化曲线（1个特征→折线图；2个特征→二维热力图），展示特征的边际效应，直观判断特征对预测的影响趋势** |
| **个体条件期望图（ICE）** | **PDP 的逐样本版本：在 PDP 图上叠加每条样本自己的条件响应曲线，识别特征效应是否在不同子群体中存在异质性（有些样本正相关、有些负相关时 ICE 会显示交叉，而 PDP 会平均掉这种差异）** |
| SHAP 摘要图 | SHAP Summary Plot（蜂群图），展示每个特征对预测的贡献分布 |
| SHAP 依赖图 | 选择单个特征，查看其 SHAP 值随特征值变化的趋势，支持叠加交互特征着色 |
| **SHAP 交互值热力图** | **选择两个特征，计算并展示二者的 SHAP 交互效应矩阵，识别哪两个特征存在显著非线性交互** |
| SHAP 瀑布图 | 选择单条样本，逐特征解释该预测结果 |
| SHAP 力图 | 单样本的 SHAP Force Plot，直观展示正/负向贡献特征 |
| 决策路径可视化 | 可选择某棵树，可视化展示其完整决策树结构（节点、分裂条件、叶值） |

#### 9.2 报告导出

|--------|------|

### 模块 10：交互式预测（重设计）

> **设计目标**：同时支持"批量文件预测"和"前端表单手动输入预测"两种模式，并提供可视化的结果展示，方便用户直接在界面上判断预测是否符合预期。

#### 10.1 批量文件预测

| 功能点 | 描述 |
|--------|------|
| 模型选择 | 从模型管理中心下拉选择要使用的模型（默认选中"生产"状态模型） |
| 导入测试集 | 上传新的 Excel/CSV 文件作为预测输入，支持拖拽 |
| 列映射检查 | 自动检测测试集列名是否与训练特征一致；不一致时逐列展示差异，支持手动拖拽映射列名 |
| **外推检测（Extrapolation Warning）** | **对标 JMP 外推警告：使用 Mahalanobis 距离或训练集分布范围检测每条待预测样本是否超出训练集的特征分布空间；超出范围的样本用橙色/红色标记，提示用户"该预测可能不可靠"，并展示哪个特征偏离最远** |
| 一键预测 | 调用已训练模型完成批量预测，进度条显示处理进度 |
| 结果预览表格 | 在表格中展示原始数据 + 预测值；分类任务额外显示各类别概率，概率以进度条形式可视化；支持按预测值排序、筛选 |
| **预测置信区间** | **回归任务：通过 Bootstrap 方法为每条预测记录生成 95% 预测区间（下限/上限），在表格中展示，不确定性大的预测用颜色区分** |
| 预测结果分布图 | 展示预测值的分布直方图（回归）或各类别预测数量柱状图（分类） |
| 高置信度/低置信度筛选 | 分类任务：一键筛选出最大类别概率 < 阈值的"不确定"样本，供人工复核 |
| 结果标注与批注 | 在预测结果表格中，用户可对某行手动标注"正确/错误/待确认"，并添加备注文字 |
| 结果导出 | 导出预测结果（含标注、置信区间）为 Excel 文件；分类任务可选择是否含各类概率列 |

#### 10.2 手动表单预测

| 功能点 | 描述 |
|--------|------|
| 特征输入表单 | 根据训练时选定的特征自动生成输入表单；数值型为数字输入框（含训练集分布范围提示：均值 ± 1σ）；类别型为下拉选择框（含训练集中出现过的所有类别值） |
| **实时外推检测** | **输入特征值时实时计算 Mahalanobis 距离，若当前输入组合超出训练集分布空间，在预测按钮旁给出橙色警示（对标 JMP 外推警告）** |
| 实时预测 | 用户填入特征值后，点击"预测"按钮，即时显示预测结果 |
| 预测结果卡片 | 分类任务：显示预测类别 + 各类概率的水平条形图；回归任务：显示预测数值 + **95% 预测区间** + 在训练集分布区间上的位置标记 |
| SHAP 单样本解释 | 预测完成后自动显示该样本的 SHAP 瀑布图，说明各特征对本次预测的贡献方向和大小 |
| **预测廓线图（What-If Profiler）** | **对标 JMP Prediction Profiler：为每个特征绘制一条响应曲线（固定其他特征时该特征与预测结果的关系），可交互拖动滑块，所有曲线实时联动；支持"优化"功能，自动搜索使预测值达到目标的特征取值组合** |
| 历史记录 | 保存每次手动预测的输入与结果，支持回溯查看和对比 |
