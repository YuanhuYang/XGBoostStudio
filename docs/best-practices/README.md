# 最佳实践示例（Best Practices）

本目录包含使用 XGBoost Studio 对**开源经典数据集**进行端到端建模的最佳实践文档。每个示例覆盖一种典型任务类型，帮助新手用户快速理解完整工作流。

## 示例列表

| 文档 | 任务类型 | 数据集规模 | 难度 | 说明 |
|------|----------|------------|------|------|
| [01-titanic-survival.md](01-titanic-survival.md) | 二分类 | 891 行 × 12 列 | 入门 | 泰坦尼克号生还预测，最经典的入门示例 |
| [02-boston-housing.md](02-boston-housing.md) | 回归 | 506 行 × 14 列 | 入门 | 波士顿房价预测，经典回归基准 |
| [03-iris-classification.md](03-iris-classification.md) | 多分类 | 150 行 × 5 列 | 入门 | 鸢尾花分类，最简单的多分类示例 |

## 使用方法

1. 按文档步骤下载数据集到本地
2. 打开 XGBoost Studio 新建项目
3. 跟随步骤一步步完成导入→分析→训练→评估
4. 对比你的结果与文档示例

## 贡献指南

欢迎新增更多典型数据集的最佳实践示例。请遵循本目录现有的结构和风格。
