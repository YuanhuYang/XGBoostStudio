# 最佳实践 2：波士顿房价预测（回归）

> **任务类型**：回归  
> **数据来源**：UCI Machine Learning Repository  
> **数据集规模**：506 行 × 14 列  
> **难度**：入门

---

## 1. 任务目标

根据波士顿地区房屋的各种特征（犯罪率、平均房间数、地段、高速公路可达性等）预测房屋的中位数价格。这是机器学习回归任务的经典基准数据集。

## 2. 下载数据集

你可以从以下地址获取数据集：

- UCI 官网：https://archive.ics.uci.edu/ml/datasets/Housing
- 直接下载 CSV 格式：https://raw.githubusercontent.com/selva86/datasets/master/BostonHousing.csv

保存文件到本地，文件名 `BostonHousing.csv`。

完整性校验（可选）：
- `BostonHousing.csv` MD5: `29AA6C5003152D07F31B3F2E7BF7AF24`

**字段说明**：

| 列名 | 说明 |
|------|------|
| crim | 城镇人均犯罪率 |
| zn | 住宅用地比例（超过 25,000 平方英尺）|
| indus | 城镇非零售商业区比例 |
| chas | Charles River 哑变量（1 表示边界，0 表示不）|
| nox | 一氧化氮浓度（每千万份）|
| rm | 平均每栋住宅房间数 |
| age | 1940 年前建成的自有住房比例 |
| dis | 到五个波士顿就业中心的加权距离 |
| rad | 高速公路可达性指数 |
| tax | 全额财产税税率 |
| ptratio | 城镇师生比例 |
| b | 1000(Bk - 0.63)^2，Bk 是黑人比例 |
| lstat | 房东中低收入群体百分比 |
| medv | 房屋中位数价格（千美元，这是**目标变量**）|

## 3. 在 XGBoost Studio 中开始

### 3.1 新建项目

1. 启动 XGBoost Studio
2. 点击「新建项目」
3. 输入项目名称：`Boston-Housing-Price`
4. 选择工作目录，点击「创建」

### 3.2 导入数据

1. 在「数据导入」页面上传 `BostonHousing.csv`
2. 查看数据预览：共 506 行 × 14 列
3. 检查缺失值：本数据集无缺失值
4. 数据质量评分应为 ~98 分（几乎完美）

## 4. 数据预处理

因为本数据集质量非常好，没有缺失值、没有异常值（统计学角度），所以：

1. 进入「智能预处理」页面
2. 系统会提示「数据质量优秀，可跳过此步」。如果你需要自定义预处理，也可以手动调整配置。
3. 直接点击「确认并继续」

## 5. 特征与划分

### 5.1 目标列选择

系统会自动推荐最后一列 `medv` 作为目标列，确认选择。任务类型自动识别为**回归**。

### 5.2 特征选择

所有 13 个特征都是输入变量，保持全选即可。

### 5.3 划分数据集

- 训练/测试比例：80/20（默认）适合
- 回归任务无需分层采样，取消勾选
- 随机种子：42 保证可复现
- 点击「划分数据集」

> 说明：不同版本的 XGBoost 或 scikit-learn 可能会导致划分结果有极其微小的差异，预期结果范围已经考虑了这种波动。

划分结果：
- 训练集：404 行
- 测试集：102 行

## 6. 参数配置（向导模式）

选择「**⚖️ 均衡推荐**」，系统会自动推荐参数：

| 参数 | 推荐值 |
|------|--------|
| n_estimators | 100 |
| max_depth | 3 |
| learning_rate | 0.1 |
| subsample | 0.8 |
| colsample_bytree | 0.8 |
| reg_lambda | 1 |

确认参数继续。

## 7. 模型训练

1. 点击「开始训练」
2. 训练很快完成（几秒钟）

预期结果（固定随机种子 42 下）：
- 测试集 RMSE ≈ 3-4（千美元，即 $3000-$4000 误差）
- 测试集 R² ≈ 0.80-0.85

如果你使用不同随机种子，结果可能会在小范围内波动，这属于正常现象。

R² = 0.85 表示模型解释了 85% 的房价方差，这在波士顿房价数据集上是不错的结果。

## 8. 模型评估（回归）

### 关键指标解读

- **RMSE（均方根误差）**：平均误差约 3-4 千美元
- **R² 得分**：0.80-0.85（满分 1.0）
- **MAE（平均绝对误差）**：约 2-3 千美元

### 残差分析

- **残差直方图**：残差应近似正态分布，均值接近 0
- **QQ 图**：点应近似落在对角线上，说明残差近似正态
- **预测值 vs 真实值散点图**：点应近似落在 y=x 直线附近

### 特征重要性

Top 3 重要特征通常是：
1. **rm**（平均房间数）- 房间越多房价越高，符合直觉
2. **lstat**（低收入比例）- 低收入比例越高房价越低
3. **nox**（一氧化氮浓度）- 环境污染越重房价越低

这完全符合经济学常识。
