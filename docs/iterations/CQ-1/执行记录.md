## 3. 备注

- **`check_wizard.py`**：须使用 **`uv run python`**，勿用系统 `python`，否则可能因 SciPy/sklearn DLL 与仓库 venv 不一致而 `ImportError`。
- **`acceptance_test.py`**：曾与 **旧进程占用 18899**（无 `/provenance` 等路由）导致 404；验收前需释放端口或停旧服务。另：原脚本末尾 Unicode 勾选符在 **GBK 控制台** 触发 `UnicodeEncodeError`，已在 [`server/tests/acceptance_test.py`](../../../server/tests/acceptance_test.py) 改为 ASCII 成功行（CQ-2 前最小修复，便于 Windows 证据链）。
