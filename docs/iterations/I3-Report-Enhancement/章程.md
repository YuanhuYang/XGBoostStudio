# 迭代章程 · I3：报表生成与数据分析能力增强

> **项目经理**：基于 JASP 竞品分析结果，增强报表生成能力，增加章节定制、模板保存、APA 格式支持。  
> **依赖**：I2 导航重构已完成。  
> **迭代 ID**：I3-Report-Enhancement。

---

## 1. 目标

| 现状 | 目标 |
|------|------|
| 报表生成默认包含所有章节，无法定制 | 支持用户选择要包含的章节，满足不同场景需求 |
| 缺少模板保存功能 | 用户可保存自己常用的报表配置为模板，下次一键生成 |
| 不支持 APA 格式 | 借鉴 JASP，增加 APA 格式对齐，便于学术发表 |

**范围冻结（本期必须做）**：
- [x] 报表生成前增加「章节选择」面板
- [x] 用户可勾选/取消勾选想要的章节
- [x] 支持保存用户勾选配置为「我的模板」
- [x] 内置「精简汇报模板」和「完整存档模板」
- [x] PDF 样式对齐 APA 格式（字体、间距、标题）
- [x] 项目模板与示例一键导入功能

**非目标（本迭代不做）**：
- 变更导航结构 → I2 已完成
- 新增数据分析算法 → 已有完整功能，不需要
- AI 辅助叙事 → G2-R1 已实现

---

## 2. 设计约束

1. **兼容性**：保持现有 API 接口不变，只增加选项参数
2. **向后兼容**：已生成的报告链接和下载不受影响
3. **默认行为**：默认勾选全部章节，保持向后兼容

## 4. 章节定义（可勾选选项）

| 章节名称 | 默认勾选 | 说明 |
|----------|----------|------|
| 数据摘要 | ✅ | 数据集基本信息、数据质量评分 |
| 数据关系分析 | ✅ | 变量相关性、与目标关系、缺失分析 |
| 参数配置 | ✅  | 模型参数配置快照 |
| 训练过程 | ✅ | 训练曲线、训练耗时总结 |
| 模型评估指标 | ✅ | 所有评估指标、置信区间 |
| 特征重要性 | ✅ | XGBoost 特征重要性排名 |
| SHAP 解释 | ✅ | SHAP 摘要图、依赖图 |
| 可视化图表汇总 | ✅ | ROC、混淆矩阵、残差图等 |
