## 四、关键技术决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 模式持久化方式 | localStorage（key: `xgbs_workflow_mode`） | 与 wizard session（`xgbs_workflow_state`）独立，互不干扰 |
| 侧边栏宽度动画 | CSS transition（不用 Ant Design collapse） | 更流畅，transition 在 Sider style 上直接加 |
| ParamLabModal 提取方式 | 完整状态提取为独立组件，SmartWorkflow 只保留 `labOpen` | 减少 SmartWorkflow 复杂度，ParamConfig 可复用同一组件 |
| 专家模式接管提示 | 3秒 auto-close Alert（`useRef` 防重复触发） | 非侵入式，不阻断工作流 |
| workflowStep 单一来源 | SmartWorkflow 用 useEffect 同步到 store | 保留 SmartWorkflow 局部 state，避免 store 驱动 UI 更新延迟 |
