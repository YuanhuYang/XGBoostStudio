## e2e_validate.py 使用说明

1. 启动后端监听 `127.0.0.1:18899`。
2. 若库中 **无模型**，脚本以退出码 0 跳过报告步骤（仅打印提示）。
3. 若有模型：依次校验 `GET /api/models` → `POST /api/reports/generate` → `GET .../download` 为 PDF。

```bash
cd server
python tests/e2e_validate.py
```
