## 一、产品定位

XGBoost Studio 是一款 **XGBoost 垂直领域的专业建模平台**，以桌面应用形式交付，**完全本地运行，无需联网**。

核心价值主张：
- 数据分析师和算法工程师无需编写代码，即可完成 XGBoost 全生命周期建模
- 内置 XGBoost 专属分析逻辑（IV/KS/PSI/单调性/泄露检测），而非通用统计分析
- 5 阶段分层调优流程，完整还原专家级调优决策逻辑，100% 可追溯
- 一键生成 12 章专业 PDF 报告，可直接用于汇报、评审、合规审计
- **三种体验模式**，覆盖业务分析师、成长期研究者、算法专家三类用户，模式间无缝切换

### 与竞品的核心差距

| 竞品 | 核心短板 | XGBoost Studio 优势 |
|------|----------|---------------------|
| JASP 0.96 | XGBoost 非原生支持，基于 R gbm 包；无调优过程追溯 | 原生 XGBoost、5 阶段完整调优记录 |
| JMP Pro 16 | 需额外插件；无 OOT 验证、无泄露检测 | 内置全链路分析，含 OOT、泄露检测 |
| H2O.ai / DataRobot | 黑盒调参，无法还原决策逻辑；私有化成本高 | 完全本地、调优全程透明可复现 |
| 代码库（XGBoost + Sklearn） | 门槛极高，90% 工作需手动完成 | 可视化全流程，零代码输出专家级结论 |

## 二、用户画像

### 业务分析师（非技术背景用户）
- **背景**：聚焦业务决策，无需掌握算法细节，熟悉 Excel 和业务数据
- **核心诉求**：上传数据 → 快速得到可靠模型 → 一份可以直接用于汇报的专业 PDF
- **产品响应**：**智能向导模式（6 步全程引导）**，每步自动推荐配置，输出自然语言结论，一键生成专业报告。**默认开启**与「模型调优」相同的参数教学卡片、预处理步骤中的「深入了解」折叠、参数实验（`client/src/utils/teachingUi.ts`：`guided` 与 `learning` 均启用；`expert` 关闭以减少干扰）

### 建模进阶用户（需在固定数据集上迭代训练与调参）
- **背景**：已完成或即将完成数据准备，希望侧栏只保留「训练 → 超参调优 → 模型管理」链路，顶栏突出**划分（Split）**上下文
- **核心诉求**：在已激活数据集上完成特征工程划分后，专注产出与对比模型、确定主模型
- **产品响应**：**模型调优模式**（应用内 `workflowMode === 'learning'`，顶栏 Segmented 显示为「调优」；localStorage 键仍为 `learning`）。进入前须已激活数据集；侧栏仅 **模型训练 / 超参数调优 / 模型管理**，顶栏**仅显示划分** Tag（点击去特征工程）

### 算法专家（数据科学家、算法工程师）
- **背景**：有 XGBoost 实战经验，需要精细调参、多版本对比、报告交付与交互预测
- **核心诉求**：完整控制建模流程，多模型横向对比，报告导出与验收，主模型上下文清晰
- **产品响应**：**专家模式（完整侧边栏）**，十页功能模块全开；顶栏**仅显示主模型** Tag（`activeModelId`，点击去模型管理）；模型注册表多维对比、Ctrl+K 命令面板；教学类入口不展示

## 三、三种体验模式

三种模式共享同一套后端与数据流，全局状态（activeDatasetId / activeSplitId / activeModelId）在切换时完整保留。

**顶栏上下文（每种模式仅一枚 Tag，其余 ID 仍保留在 store 中但不展示）**：向导 → **数据集**；模型调优 → **划分**；专家 → **主模型**。

```
┌────────────────────────────────────────────────────────────────────────┐
│  模式切换器（Header 右侧 Segmented）                                     │
│  🎯 向导   🔧 调优（模型调优）   ⚙️ 专家                                   │
└────────────────────────────────────────────────────────────────────────┘
         │                   │                   │
    侧边栏：极简             侧边栏：训练链路        侧边栏：完整 10 模块
    仅智能向导              训练/调优/管理         按功能分组 + 状态徽章
    步骤进度圆点            「已学」角标           220px 宽展开
         │                   │                   │
    6 步引导 + 默认教学    划分顶栏 + 专注建模   全模块 + 主模型顶栏
    自动推荐 + 参数实验    需先激活数据集       对比/报告/预测/Ctrl+K
    自然语言结论           确定主模型           无教学类入口
```

### 模式切换机制（v0.4 新增）

- **切换器位置**：顶部 Header 右侧，Ant Design Segmented 三态控件
- **localStorage 持久化**：刷新或重启后恢复上次使用的模式
- **训练中保护**：SSE 训练进行中切换模式时弹出确认弹窗，训练不中断
- **首次引导**：每种模式首次进入时展示 `ModeOnboardingModal`，介绍该模式特性
- **向导模式拦截**：向导模式下点击侧边栏非向导项，弹出「是否暂离向导？已保存进度」确认
- **进入模型调优前校验**：切换到「调优」时若尚未激活数据集，弹出提示并保持当前模式
- **向导→专家桥接**：向导第 6 步结果页可一键「切换到专家模式继续分析」

### 智能向导 6 步流程（向导模式专属）

```
Step 0: 数据准备       → 上传/选择数据集，查看质量评分（0-100），推断任务类型
Step 1: 数据分析       → 数据质量与任务类型展示、AI 预处理建议卡片（与 SmartWorkflow 实际页面对齐）
Step 2: 数据划分       → 目标列确认与训练/测试划分
Step 3: 参数配置       → 基于划分推荐 XGBoost 参数（可切换预设/高级参数）
Step 4: 一键训练       → 训练流水线（SSE）+ 评估与自然语言小结
Step 5: 结果总结       → 导出报告、历史记录、切换专家模式等
```

**Step 0 增强：全自动建模（AutoML）**  
在同一页提供可选入口：在**固定试验预算**内自动完成「目标列启发式 → 划分 → 规则推荐参数 → 多候选训练 → 按验证指标与过拟合等级惩罚排序」。产出多条 `Model` 记录，用户可用 Radio 切换主模型并跳转「结果总结」。**不承诺全局最优**，详见 [`08-automl-wizard.md`](08-automl-wizard.md)。

**命令行模式（开发者 / 脚本 / 与 UI 并行）**  
源码用户在 `server` 目录可运行 **`python -m cli.main`**（包名 `xs-studio`）：默认启动后端子进程并进入交互式 REPL，或通过 **`run <数据文件路径>`** 非交互完成上传与 AutoML；与浏览器中的前端共用同一 API 与数据库，并可用 `urls` / `run` 输出的**深链**打开前端以继续看模型对比与报告。说明见 [`08-automl-wizard.md`](08-automl-wizard.md) §2.2 与 [xs-studio CLI 指南](../guides/xs-studio-cli.md)。

### 教学增强能力（智能向导 + 模型调优默认；专家模式关闭）

由 `showTeachingUi(workflowMode)` 统一控制（`guided` 与 `learning` 为 true，`expert` 为 false）。

| 组件 | 位置 | 功能 |
|------|------|------|
| `LearningPanel` | `ParamExplainCard` 内 | 算法直觉 + 调大/调小效果 + 过拟合风险色阶条 |
| `ParamLabModal` | 参数配置页 / 智能向导 Step 3 | 对比两组参数配置的训练曲线和评估指标 |
| 收敛解释卡 | 模型训练页训练完成后 | 基于 train/val loss 差值自动生成收敛分析文字 |
| 概念卡片 Popover | 特征分析页 IV/KS/PSI Tab | 指标含义、判断阈值、计算原理解释 |
| 指标解释 Tooltip | 模型评估页各指标卡片 | AUC/KS/F1/R²/RMSE 等指标的自然语言解释 |
| 预处理「深入了解」 | 智能向导数据分析步 | 预处理建议卡片内可折叠展开统计原理说明 |

### 专家模式专属功能（v0.4 新增）

| 功能 | 位置 | 说明 |
|------|------|------|
| 完整 10 模块菜单 | 侧边栏 | 宽 220px，按功能分组，菜单项带绿色状态圆点 |
| 顶栏主模型 Tag | Header | 仅展示当前 `activeModelId`，便于串联评估、报告与预测 |
| AUC/KS 差值列 | 模型管理对比表 | 以第一个模型为基准，显示各模型指标相对差值 |
| 调优轨迹最优点标记 | 超参数调优轨迹图 | 金色 Pin 标注全局最优 trial 位置 |
| Ctrl+K 命令面板 | 全局（专家模式） | 搜索所有页面，键盘快速跳转 |
| 接管提示条 | 进入专家模式时 | 显示「已接管向导进度：数据集/划分/模型」3 秒提示 |

## 四、功能模块总览

| 模块序号 | 模块名称 | 核心功能 |
|----------|----------|----------|
| 0 | 智能向导 | 三模式入口、6步向导流程、历史记录恢复、**Step 0 全自动建模（AutoML）** |
| 1 | 数据导入与预览 | CSV/XLSX 导入、质量评分、缺失值/异常值/重复行处理 |
| 2 | 特征分析 | IV/KS/PSI/单调性分析、泄露检测、标签专项分析、VIF、PCA |
| 3 | 特征工程与选择 | 编码/缩放/Box-Cox/选择、时间序列与随机划分 |
| 4 | 参数配置 | 参数推荐、自定义配置、参数说明教学卡片、参数实验室 |
| 5 | 模型训练 | 实时 SSE 进度、K折交叉验证、早停机制、收敛解释卡（向导/模型调优默认） |
| 6 | 模型评估 | 混淆矩阵/ROC/PR/校准、PDP/ICE、OOT验证、鲁棒性测试、坏样本诊断 |
| 7 | 超参数调优 | 5阶段分层调优、调优轨迹图（含最优点标记）、各阶段参数锁定与传递 |
| 8 | 模型管理 | 模型注册表、多维对比（含差值列）、McNemar 检验、版本管理 |
| 9 | 分析报告 | 12章 PDF 报告、4种预设模板、品牌定制 |
| 10 | 交互式预测 | 文件批量预测 + 表单手动预测、SHAP 单样本解释 |
